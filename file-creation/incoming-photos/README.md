# incoming-photos — תיקיית קליטת תמונות מגירות

לכאן מכניסים **תמונות של מגירות** (לעגלות שיש להן רק תמונות, בלי סקיצות/מק"טים).

## איך לשלוח
1. צלם את המגירה מלמעלה, באור טוב, כך שכל הכלים נראים.
2. שמור את התמונה עם שם שמספר לי לאן היא שייכת, למשל:
   - `C03-A1.jpg`  → עגלה C03, מגירה A1
   - אם אין עדיין מזהה — כל שם, ותכתוב לי "זו עגלה X מגירה Y".
3. אפשר גם פשוט לשלוח לי את התמונה כאן בצ'אט.

## מה קורה אחר כך
Claude מזהה את הכלים בתמונה → מכין שורות (תיאור + כמות) → שואל אותך על מה שלא ברור
(מק"ט יצרן בדרך כלל לא נראה בתמונה, אז תשלים אותו) → מייצר קובץ Excel/CSV מוכן לטעינה
לתוך התוכנה. הפלט נשמר ב-`file-creation/output-excel/`.
